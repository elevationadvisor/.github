# Elevation Advisor

Elevation Advisor is a construction business management platform for design-build firms.
Our mission: **Get your life back.**

## What We’re Building
Elevation Advisor helps owners:
- Price work correctly
- Schedule crews intelligently
- Understand profitability in real time
- Reduce cognitive load through AI-driven assistance

## Core Repositories
- **elevation-next-js** – Web application (Next.js)
- **elevation-api / altitude.aws** – Backend services (AWS)
- **elevation-strapi** – CMS and content management
- **altitude.ReactNative** – Mobile application
- **altitude.kb** – Knowledge base and documentation

## Tech Stack (High-Level)
- Frontend: React, Next.js
- Backend: AWS (Lambda, DynamoDB, Redshift)
- Mobile: React Native
- AI: AWS Bedrock (Claude)
- Infra: Cloudflare, S3

## How We Work
- Incremental, testable features
- Clear success criteria per roadmap phase
- Documentation-first for complex logic

Most documentation lives in `altitude.kb`.
